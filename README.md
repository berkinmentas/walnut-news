# Walnut News Scraper

News scraping service for Walnut Portal project. Built with Node.js and Puppeteer.

## Requirements
- Node.js LTS
- Docker
- Docker Compose

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file from `.env.example`
4. Run: `npm start`

## Docker
Run with Docker Compose:
```docker-compose up```
